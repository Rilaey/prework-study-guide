CSS

  • Enter your CSS notes here
  • the margin property indicates how much space we want around the outside of the element.
  • when we set the margin to 0 it sets no extra space around the element.
  • don't get padding confused with the margin property. when we use padding, it adds space inside of the element. when we increase padding we make the entire element bigger.
  • we can use "*" to apply whatever css property we use to all HTML elements.
  • To designate a class in css you put a period before naming the class you plan to assign.
